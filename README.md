## 📌 Project Overview \| Tổng quan dự án

**EN**\
Goodwill Vietnam is a web-based charity platform designed to connect
**donors, recipients, and administrators** in a centralized system. It
supports the full donation lifecycle:\
**Submission → Approval → Warehouse Intake → Distribution**

**VIE**\
Goodwill Vietnam là nền tảng thiện nguyện giúp kết nối **người quyên
góp, người nhận và quản trị viên** trên một hệ thống tập trung.\
**Gửi → Duyệt → Nhập kho → Phân phối**

## 🚀 Features \| Tính năng

-   Donation Management \| Quản lý quyên góp\
-   Tracking System \| Theo dõi trạng thái\

-   Campaign Management \| Chiến dịch\
-   Admin Dashboard \| Quản trị\
-   Charity Shop \| Shop thiện nguyện
