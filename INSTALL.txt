II. CÀI ĐẶT XAMPP
────────────────────────────────────────────────────────────────────────────

1. Tải XAMPP:
   → Truy cập: https://www.apachefriends.org/
   → Chọn phiên bản Windows
   → Tải và cài đặt

2. Cài đặt XAMPP:
   → Chạy file cài đặt
   → Chọn thư mục cài đặt: C:\xampp
   → Chọn các component: Apache, MySQL, PHP, phpMyAdmin
   → Hoàn tất cài đặt

3. Khởi động XAMPP:
   → Mở XAMPP Control Panel
   → Click "Start" cho Apache
   → Click "Start" cho MySQL
   → Đợi đến khi hiển thị màu xanh


III. CÀI ĐẶT DỰ ÁN
────────────────────────────────────────────────────────────────────────────

BƯỚC 1: COPY FILE DỰ ÁN
─────────────────────────
1. Mở thư mục: C:\xampp\htdocs
2. Tạo thư mục mới: "Cap 1 - 2"
3. Copy toàn bộ file dự án vào thư mục này
   
   Cấu trúc sau khi copy:
   C:\xampp\htdocs\Cap 1 - 2\
   ├── admin/
   ├── api/
   ├── assets/
   ├── config/
   ├── database/
   ├── includes/
   ├── uploads/
   ├── index.php
   ├── login.php
   └── ...

BƯỚC 2: TẠO CƠ SỞ DỮ LIỆU
───────────────────────────
1. Mở trình duyệt
2. Truy cập: http://localhost/phpmyadmin
3. Click tab "New" (hoặc "Mới")
4. Nhập tên database: goodwill_vietnam
5. Chọn Collation: utf8mb4_unicode_ci
6. Click "Create"

BƯỚC 3: IMPORT DỮ LIỆU
───────────────────────
1. Trong phpMyAdmin, chọn database "goodwill_vietnam"
2. Click tab "Import" (hoặc "Nhập")
3. Click "Choose File" (hoặc "Chọn file")
4. Chọn file: database/schema.sql
5. Click "Go" (hoặc "Thực hiện")
6. Đợi import hoàn tất
7. Lặp lại với file: database/update_schema.sql

   ⚠️ QUAN TRỌNG: Import đúng thứ tự:
      1. schema.sql (trước)
      2. update_schema.sql (sau)

BƯỚC 4: CẤU HÌNH KẾT NỐI
──────────────────────────
1. Mở file: config/database.php
2. Kiểm tra cấu hình (mặc định):
   
   DB_HOST: localhost
   DB_NAME: goodwill_vietnam
   DB_USER: root
   DB_PASS: (để trống)

3. Nếu MySQL của bạn có mật khẩu:
   → Sửa DB_PASS thành mật khẩu của bạn

4. Lưu file

BƯỚC 5: TẠO THƯ MỤC UPLOAD
────────────────────────────
1. Kiểm tra thư mục: uploads/
2. Tạo thư mục con: uploads/donations/
3. Phân quyền ghi cho thư mục (Windows):
   → Chuột phải uploads/ → Properties
   → Tab Security → Edit
   → Chọn Users → Tick Full Control
   → Click OK

V. XỬ LÝ LỖI THƯỜNG GẶP
────────────────────────────────────────────────────────────────────────────

❌ LỖI: "Connection failed" hoặc "Database connection error"
───────────────────────────────────────────────────────────
NGUYÊN NHÂN: Không kết nối được database
CÁCH SỬA:
   1. Kiểm tra MySQL đã chạy trong XAMPP Control Panel
   2. Kiểm tra file config/database.php
   3. Kiểm tra tên database: goodwill_vietnam
   4. Restart Apache và MySQL

❌ LỖI: "Table doesn't exist"
───────────────────────────────
NGUYÊN NHÂN: Chưa import database hoặc import thiếu
CÁCH SỬA:
   1. Vào phpMyAdmin
   2. Xóa database goodwill_vietnam (nếu có)
   3. Tạo lại database mới
   4. Import lại schema.sql
   5. Import lại update_schema.sql

❌ LỖI: "Failed to move uploaded file"
───────────────────────────────────────
NGUYÊN NHÂN: Không có quyền ghi vào thư mục uploads
CÁCH SỬA:
   1. Kiểm tra thư mục uploads/ tồn tại
   2. Phân quyền Full Control cho thư mục
   3. Restart Apache

❌ LỖI: Trang trắng, không hiển thị gì
───────────────────────────────────────
NGUYÊN NHÂN: Lỗi PHP không hiển thị
CÁCH SỬA:
   1. Mở file php.ini: C:\xampp\php\php.ini
   2. Tìm: display_errors = Off
   3. Sửa thành: display_errors = On
   4. Lưu và restart Apache
   5. Xem lỗi cụ thể trên màn hình

❌ LỖI: "Port 80 is already in use"
────────────────────────────────────
NGUYÊN NHÂN: Port 80 đang được sử dụng (Skype, IIS...)
CÁCH SỬA:
   1. Tắt Skype hoặc ứng dụng sử dụng port 80
   2. HOẶC đổi port Apache:
      - Mở file: C:\xampp\apache\conf\httpd.conf
      - Tìm: Listen 80
      - Sửa thành: Listen 8080
      - Truy cập: http://localhost:8080/

❌ LỖI: CSS/JS không load
──────────────────────────
NGUYÊN NHÂN: Đường dẫn sai hoặc file không tồn tại
CÁCH SỬA:
   1. Kiểm tra thư mục assets/ tồn tại
   2. Kiểm tra thư mục assets/css/ và assets/js/
   3. Clear cache trình duyệt (Ctrl + F5)
